# Mark&Read

## Problem
Students waste their most of time in studying irrevalnt content and large text PDFs. 

## Solution
Mark&Read uses AI (Gemini 3.0) to analyze PDFs and generate an high-probability study version which will be revelant for their exam purposes, presentation and 
extracting content from their reserch papers , help them minimizing time time in reading large PDF texts. 

## Tech Stack
- FastAPI
- Gemini API
- PyMuPDF
- Python

## Features
- Ai- based importance analysis
-Block-level higlighting
-Ready-to-revise Pdf output

## How to Run
1. Install dependencies
 pip install -r requirements.txt
2. Start server
 uvicorn main:app --reload
3. Open Browser
 Open /docs
